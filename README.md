## Development

Install sst3 dependencies with `pnpm`.

```bash
  pnpm install
```

Run sst dev in root project.

```bash
pnpm dev
```

Run web app in `packages/web` directory.

```bash
pnpm dev
```

Then your web app will be running on `http://localhost:3000`.
